# DQN Learning Map

## What to learn in order
1. Q-network
2. replay buffer
3. agent logic
4. training loop
5. full implementation
6. environment behavior through CartPole

## What DQN helps you understand
- value-based learning
- bootstrap targets
- replay for decorrelation
- target network for stability
- epsilon-greedy exploration

## Main environment fit
DQN is easiest to understand first in environments with:
- discrete action spaces
- relatively clear feedback
- simpler control structure

## Watch out for
- wrong target computation
- replay buffer misuse
- epsilon decay too aggressive
- interpreting early noise as total failure
